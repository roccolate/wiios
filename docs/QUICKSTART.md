# Quickstart

## 1) Build
```bash
cd wiios
make clean && make
```

## 2) Package
```bash
cd wiios
make package
```

## 3) Copiar a SD
Copiar `dist/apps/wiios` a:
- `sd:/apps/wiios`

## 4) Ejecutar
- En Wii real: abrir Homebrew Channel y correr WiiOS.
- En Dolphin: abrir `dist/apps/wiios/boot.dol`.

## 5) Controles
- A: OK
- B: Back
- Left/Right: navegar foco/selección
- Home o Start: salir
